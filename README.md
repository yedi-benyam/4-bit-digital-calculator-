# 4-bit-digital-calculator

## Overview
A combinational logic circuit designed by proteus to perform basic arithmetic and logic operations using digital logic gates.

## Operations
- Addition
- Subtraction
- Comparison

## Tools Used
- Proteus Design Suite
- Digital Logic Design (DLD)
- Logic Gates (AND, OR, XOR, NOT)

## Design Methodology
- Truth table creation
- K-map simplification
- Logic gate implementation
- Circuit simulation in Proteus

## Files
- Proteus circuit design
- Technical report (PDF)
- Simulation screenshots

## Key Learning Outcomes
- Combinational logic design
- Circuit simplification using K-maps
- Digital system simulation
